📋 Testing Instructions

  1. Open Chrome DevTools: Press F12
  2. Go to Performance tab
  3. Click Record (red circle)
  4. Perform interaction (click button below or on main site)
  5. Stop recording
  6. Check "Interaction" section in timeline

Mobile Testing: Enable Device Toolbar (Ctrl+Shift+M) → Set to "iPhone 12 Pro" → Enable "4x slowdown" in Performance tab
